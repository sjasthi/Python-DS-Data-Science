üêï Let's Create a Pet Class!

Here's how we define a class in Python:

class Pet: def __init__(self, name, animal_type, age): self.name = name self.animal_type = animal_type self.age = age self.happiness = 50 def feed(self): self.happiness += 10 return f"{self.name} is happy! Happiness: {self.happiness}" def play(self): self.happiness += 15 return f"{self.name} loves playing! Happiness: {self.happiness}" def introduce(self): return f"Hi! I'm {self.name}, a {self.age} year old {self.animal_type}!"

Key Parts Explained:

‚Ä¢ __init__ - This is the constructor, it runs when we create a new pet

‚Ä¢ self - This refers to the specific pet object we're working with

‚Ä¢ Methods like feed() and play() are things our pet can do

üéâ What We Learned!

Classes are like blueprints that define:

‚Ä¢ What actions an object can perform (methods like feed, play)


Objects are individual instances created from a class:

‚Ä¢ Each object has its own unique data
