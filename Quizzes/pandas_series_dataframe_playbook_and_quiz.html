üìä Understanding Pandas Series

üìùWhat is a Series?

A Series is like a smart list! It's a one-dimensional array that can hold any type of data (numbers, text, dates, etc.). Each item in a Series has an index (like a label or position number).

üéÆ Real-World Example: Video Game Scores

Imagine you're tracking your scores in your favorite game across different levels:

Index (Level) Score
Level 11500
Level 22300
Level 31800
Level 42700

This is a perfect example of a Series!

üíª Creating a Series in Python

import pandas as pd # Method 1: From a list scores = pd.Series([1500, 2300, 1800, 2700]) print(scores) # Method 2: With custom index (labels) scores = pd.Series( [1500, 2300, 1800, 2700], index=['Level 1', 'Level 2', 'Level 3', 'Level 4'] ) print(scores)

üéØ Interactive Example: Create Your Own Series

Let's create a Series of your favorite subjects and their scores!

üîç Accessing Series Data

# Access by index position (like a list) print(scores[0]) # First score: 1500 # Access by label (if you have custom index) print(scores['Level 1']) # Score for Level 1: 1500 # Get multiple values print(scores[[0, 2]]) # Scores at positions 0 and 2 # Slice the Series print(scores[1:3]) # Scores from position 1 to 2

üé® What Can You Do With a Series?

üìà Math Operations

# Add bonus points scores + 100 # Calculate average scores.mean() # Find max score scores.max()

üîé Analysis

# Get statistics scores.describe() # Sort values scores.sort_values() # Count values scores.count()

üìã Understanding Pandas DataFrames

üìöWhat is a DataFrame?

A DataFrame is like a table or spreadsheet! It has rows and columns, where each column is actually a Series. Think of it as multiple Series put together side by side.

üè´ Real-World Example: Class Gradebook

Imagine your teacher's gradebook with student names, test scores, and grades:

Name Math Science English Grade
Alice958892A
Bob879085B
Charlie928588B
Diana889295A

This is a DataFrame with 4 rows and 5 columns!

üíª Creating a DataFrame in Python

import pandas as pd # Method 1: From a dictionary data = { 'Name': ['Alice', 'Bob', 'Charlie', 'Diana'], 'Math': [95, 87, 92, 88], 'Science': [88, 90, 85, 92], 'English': [92, 85, 88, 95] } df = pd.DataFrame(data) print(df) # Method 2: From a list of lists data = [ ['Alice', 95, 88, 92], ['Bob', 87, 90, 85] ] df = pd.DataFrame(data, columns=['Name', 'Math', 'Science', 'English']) print(df)

üéØ Interactive Example: Build Your Own DataFrame

Create a DataFrame of your friends and their favorite activities!

üîç Accessing DataFrame Data

# Get a column (returns a Series) print(df['Math']) # All math scores - this is a Series! # Get multiple columns (returns a DataFrame) print(df[['Name', 'Math']]) # Returns a smaller DataFrame # Get a row by position (iloc) - returns a Series! print(df.iloc[0]) # First row as a Series # Get a row by index (loc) - also returns a Series! print(df.loc[0]) # Row at index 0 as a Series # Get a specific value (returns a single value) print(df.loc[0, 'Math']) # Alice's math score: 95

üéØImportant: Rows and Columns are Both Series!

Here's a key concept: Both columns AND rows in a DataFrame are Series, but they work differently:

üìä Column = Series (Usually Same Type)

When you get a column, all values are typically the same data type:

# All numbers (integers) math_scores = df['Math'] # [95, 87, 92, 88] # dtype: int64

Index: Student names or row numbers
Values: All same type (numbers)

üìù Row = Series (Mixed Types)

When you get a row, values are often mixed data types:

# Mixed: string and numbers first_student = df.iloc[0] # Name: 'Alice' (string) # Math: 95 (int) # Science: 88 (int) # dtype: object (mixed)

Index: Column names
Values: Mixed types (text + numbers)

üí° Key Insight: A Series is just a 1-dimensional labeled array. It doesn't care if the data types are all the same or mixed!

‚Ä¢ Columns are usually the same type (all numbers, all text, etc.)
‚Ä¢ Rows are usually mixed types (name + scores + grades)
‚Ä¢ Both are Series objects in Pandas!
