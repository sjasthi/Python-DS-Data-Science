🌐 What Is Web Scraping?

The Big Picture

Every time you visit a website, your browser sends a request to a server. The server sends back an HTML file. Your browser renders that HTML into the pretty page you see.

Web scraping means doing the same thing — but with Python. Instead of a browser reading the HTML, your Python code reads it and extracts the data you care about.

📦 Setup: Two Libraries Working Together

Why two libraries? requests fetches the raw HTML from the internet. beautifulsoup4 then parses and navigates that HTML. They work as a team!
Python ▶ Run 
# Step 1 — Parse an HTML string (no internet needed!)
from bs4 import BeautifulSoup

html = """
<html>
  <head><title>My Sports Page</title></head>
  <body>
    <h1>Top NBA Players 2025</h1>
    <p class="intro">This season's scoring leaders.</p>
    <p>Data is updated weekly.</p>
  </body>
</html>
"""

soup = BeautifulSoup(html, 'html.parser')

print("Title:", soup.title.text)
print("H1:   ", soup.h1.text)
print("First P:", soup.p.text)
print()
print("✅ soup object created successfully!")

💡 Key Concept: soup as a Map

Think of soup like a searchable map of an HTML document. Once you create it, you can jump to any tag instantly — like using Ctrl+F to find things in a document, but powered by Python.

Python — Navigating Tags ▶ Run 
from bs4 import BeautifulSoup

html = """
<html><head><title>Player Stats</title></head>
<body>
  <h1>NBA Scoring Leaders</h1>
  <p class="intro">Season 2024-25</p>
  <h2>Western Conference</h2>
  <h2>Eastern Conference</h2>
</body></html>
"""

soup = BeautifulSoup(html, 'html.parser')

# Accessing tags directly
print("soup.title.text  =", soup.title.text)
print("soup.h1.text     =", soup.h1.text)
print("soup.h2.text     =", soup.h2.text)  # only FIRST h2
print("soup.p.text      =", soup.p.text)

🔍 .find() vs .find_all()

MethodReturnsUse when…
.find('tag')The first matching tagYou want one specific element
.find_all('tag')A list of all matchesYou want multiple elements
Python — .find() vs .find_all() ▶ Run 
from bs4 import BeautifulSoup

html = """
<html><body>
  <h2 class="sport">Basketball</h2>
  <h2 class="sport">Soccer</h2>
  <h2 class="sport">Baseball</h2>
  <p class="intro">Sports are great for teamwork!</p>
  <p class="detail">Basketball was invented in 1891.</p>
</body></html>
"""

soup = BeautifulSoup(html, 'html.parser')

# .find() — gets only the FIRST match
first_sport = soup.find('h2')
print(".find('h2'):", first_sport.text)
print()

# .find_all() — gets ALL matches as a list
all_sports = soup.find_all('h2')
print(f".find_all('h2') found {len(all_sports)} items:")
for tag in all_sports:
    print("  -", tag.text)
print()

# Search by class_ (note the underscore!)
intro = soup.find('p', class_='intro')
print("find by class_='intro':", intro.text)

🏷️ Extracting Text and Attributes

Two ways to get attribute values:
tag['href'] — crashes if the attribute is missing
tag.get('href') — returns None if missing (safer!)
Python — Text & Attributes ▶ Run 
from bs4 import BeautifulSoup

html = """
<html><body>
  <a href="https://www.nba.com" class="league" data-sport="basketball">NBA</a>
  <a href="https://www.mlb.com" class="league" data-sport="baseball">MLB</a>
  <a href="https://www.nfl.com" class="league" data-sport="football">NFL</a>
</body></html>
"""

soup = BeautifulSoup(html, 'html.parser')

# Loop through all links
print(f"{'Text':<8} {'URL':<30} {'Sport'}")
print("-" * 55)
for a in soup.find_all('a'):
    text  = a.get_text(strip=True)
    url   = a.get('href')
    sport = a.get('data-sport')
    print(f"{text:<8} {url:<30} {sport}")

🃏 The Cards Pattern

Many websites use repeated card blocks — the same div class used for each item in a list (players, products, news articles). Finding all cards at once is one of the most common scraping patterns.

Python — Scraping Cards ▶ Run 
from bs4 import BeautifulSoup

html = """
<html><body>
<div class="player-card">
  <h3 class="name">Shai Gilgeous-Alexander</h3>
  <span class="team">OKC Thunder</span>
  <span class="ppg">32.7 PPG</span>
</div>
<div class="player-card">
  <h3 class="name">Giannis Antetokounmpo</h3>
  <span class="team">Milwaukee Bucks</span>
  <span class="ppg">30.4 PPG</span>
</div>
<div class="player-card">
  <h3 class="name">Luka Doncic</h3>
  <span class="team">LA Lakers</span>
  <span class="ppg">28.1 PPG</span>
</div>
</body></html>
"""

soup = BeautifulSoup(html, 'html.parser')
cards = soup.find_all('div', class_='player-card')
print(f"Found {len(cards)} player cards\n")

for card in cards:
    name = card.find('h3', class_='name').get_text(strip=True)
    team = card.find('span', class_='team').get_text(strip=True)
    ppg  = card.find('span', class_='ppg').get_text(strip=True)
    print(f"{name} | {team} | {ppg}")

📊 Scraping Tables into DataFrames

Why Tables?

HTML tables are the #1 scraping target for data scientists. Sports stats, financial data, Wikipedia data — it's all in tables. The pattern is always the same:

STEP 1Find the table with soup.find('table')
↓
STEP 2Extract headers from <th> tags
↓
STEP 3Loop through rows (skip row 0 — it's the header!)
↓
STEP 4Build a Pandas DataFrame
Python — Full Table Scraping Pattern ▶ Run 
from bs4 import BeautifulSoup
import pandas as pd

html = """
<table id="players">
  <tr><th>Rank</th><th>Player</th><th>Team</th><th>PPG</th></tr>
  <tr><td>1</td><td>SGA</td><td>OKC Thunder</td><td>32.7</td></tr>
  <tr><td>2</td><td>Giannis</td><td>Milwaukee Bucks</td><td>30.4</td></tr>
  <tr><td>3</td><td>Luka</td><td>LA Lakers</td><td>28.1</td></tr>
  <tr><td>4</td><td>KD</td><td>Phoenix Suns</td><td>27.3</td></tr>
</table>
"""

soup = BeautifulSoup(html, 'html.parser')

# Step 1 - Find the table
table = soup.find('table', id='players')

# Step 2 - Extract headers
headers = [th.get_text(strip=True) for th in table.find_all('th')]
print("Headers:", headers)

# Step 3 - Extract rows (skip row 0!)
rows = []
for tr in table.find_all('tr')[1:]:
    cells = tr.find_all('td')
    row = [c.get_text(strip=True) for c in cells]
    if row:
        rows.append(row)

# Step 4 - Build DataFrame
df = pd.DataFrame(rows, columns=headers)
df['Rank'] = df['Rank'].astype(int)
df['PPG']  = df['PPG'].astype(float)

print("\nDataFrame:")
print(df.to_string(index=False))
print()
print("Average PPG:", df['PPG'].mean())

⚡ pd.read_html() Shortcut

The Fast Way — When it Works

Pandas has a built-in function that reads tables directly from a URL. It's great for Wikipedia and simple sites.

Limitation: pd.read_html() can't filter by class or id, and it doesn't work on all sites. For complex pages, the manual Beautiful Soup approach gives you more control.

Python — pd.read_html() (Run in Colab) ▶ Open in Colab 
import pandas as pd

# Read ALL tables from a Wikipedia page
url = "https://en.wikipedia.org/wiki/List_of_highest-grossing_films"
tables = pd.read_html(url)

print(f"Found {len(tables)} tables on the page")

# Get the first table
df = tables[0]
print(df.head(5))
💡 Pro Tip: When scraping tables, always check your data types! Numbers like PPG often come in as strings. Use pd.to_numeric(df['col']) or .astype(float) to convert them.

⚖️ Ethical Web Scraping

Why Ethics Matter

Web scraping is a powerful tool, but it can cause real harm if used carelessly. Servers cost money to run. Scraping too fast can crash a website for other users. Some data is private and shouldn't be collected. Being a responsible scraper is part of being a professional data scientist.

RuleWhy It Matters
Check robots.txtVisit https://example.com/robots.txt — it tells bots which pages they may not scrape
Add delaysUse time.sleep(1) between requests — don't overload the server
Respect Terms of ServiceMany sites legally prohibit scraping in their ToS
Don't scrape personal dataAvoid collecting names, emails, or private information
Use APIs when availableIf a site offers an API, use it — it's faster, cleaner, and officially allowed
Identify your botSet a User-Agent header so the site knows who is scraping

😴 Polite Scraping with time.sleep()

Python — Ethical Multi-Page Scraping ▶ Open in Colab 
import requests
from bs4 import BeautifulSoup
import time

urls = [
    "https://example.com/page1",
    "https://example.com/page2",
    "https://example.com/page3"
]

for url in urls:
    response = requests.get(url)

    if response.status_code == 200:
        soup = BeautifulSoup(response.text, 'html.parser')
        # ... extract data ...
        print(f"✅ Scraped: {url}")
    else:
        print(f"❌ Failed ({response.status_code}): {url}")

    time.sleep(1)    # wait 1 second before the next request
