Matplotlib Visualizations

Choosing the right chart for quantitative & qualitative variables

Quantitative Variables

Variables you can count or measure with numbers

What is a quantitative variable?
A quantitative (numerical) variable represents quantities that you can count or measure. It makes sense to add, subtract, or average these values. Examples: height, temperature, test scores, age, number of siblings.

Quantitative variables come in two flavors:

ContinuousCan take any value within a range — height (5.3 ft), temperature (98.6°F), time (2.75 hours)
DiscreteWhole number counts — number of pets (3), goals scored (2), apps on your phone (47)

When we visualize a single quantitative variable, we want to understand its distribution — how the values spread out, where they cluster, and whether there are outliers.

📶 Histogram

A histogram divides the range of values into equal-width bins and shows how many data points fall into each bin. It reveals the shape of a distribution — is it symmetric? Skewed? Where do most values land?

When to use: You have one list of numbers and want to see how frequently different values appear. Great for test scores, heights, ages, temperatures.
Python
import matplotlib.pyplot as plt # Test scores for a class scores = [72, 85, 90, 67, 88, 95, 78, 82, 91, 74, 69, 88, 93, 77, 81, 86, 79, 92, 84, 70] plt.hist(scores, bins=5, color='steelblue', edgecolor='white') plt.title('Distribution of Test Scores') plt.xlabel('Score') plt.ylabel('Frequency') plt.show()

🔑 Key Parameters

binsNumber of intervals to divide the data into. More bins → more detail. Try different values!
colorFill color of the bars (e.g., 'steelblue', 'coral')
edgecolorColor of the border around each bar — use 'white' to visually separate bars

📦 Box Plot

A box plot (box-and-whisker plot) provides a 5-number summary at a glance: minimum, Q1 (25th percentile), median (Q2), Q3 (75th percentile), and maximum. Dots beyond the whiskers are outliers.

When to use: You want to quickly see the center, spread, and outliers of your data. Also excellent for comparing distributions of different groups side by side.
Python
import matplotlib.pyplot as plt # Daily step counts for a week steps = [4500, 7200, 6800, 8100, 3200, 9500, 7000] plt.boxplot(steps, vert=True, patch_artist=True, boxprops=dict(facecolor='lightcoral')) plt.title('Daily Step Counts') plt.ylabel('Steps') plt.show()

🔑 Key Parameters

vertTrue for vertical box, False for horizontal
patch_artistSet to True to fill the box with color
boxpropsDictionary to customize the box appearance, e.g. facecolor

📈 Line Chart

A line chart connects data points in order, showing how a quantitative variable changes over time or across a sequence. The x-axis is usually a time period (days, months, years).

When to use: You have data collected over time (a time series) and want to see trends — is it going up? Down? Staying flat?
Python
import matplotlib.pyplot as plt # Monthly average temperatures (°F) months = ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun'] temps = [30, 35, 45, 55, 65, 75] plt.plot(months, temps, marker='o', color='tomato', linewidth=2) plt.title('Average Monthly Temperatures') plt.xlabel('Month') plt.ylabel('Temperature (°F)') plt.show()

🔑 Key Parameters

markerShape at each data point — 'o' circle, 's' square, '^' triangle
linewidthThickness of the connecting line
colorColor of the line (e.g., 'tomato', '#3498db')

Qualitative Variables

Also called categorical or nominal variables

What is a qualitative variable?
A qualitative (categorical / nominal) variable represents categories or groups. You can name them but you cannot meaningfully add or average them. Examples: favorite color, type of pet, city name, grade letter (A/B/C), genre of music.

Qualitative variables come in two flavors:

NominalNo natural order — colors, pet types, music genres, sports teams
OrdinalHave a meaningful order — grade letters (A > B > C), satisfaction rating (Happy > Neutral > Sad)

When we visualize a qualitative variable, we want to count how many items fall into each category and compare those counts.

📊 Bar Chart

A bar chart displays each category as a separate bar, with the height (or length) of the bar representing the count or value for that category. Categories go on one axis, values on the other.

When to use: You want to compare counts or values across different categories. Great for "Which is the most/least popular?" questions.
Python — Vertical Bar Chart
import matplotlib.pyplot as plt # Favorite sports in class sports = ['Soccer', 'Basketball', 'Baseball', 'Tennis', 'Swimming'] votes = [12, 9, 6, 4, 7] plt.bar(sports, votes, color='teal') plt.title('Favorite Sports in Our Class') plt.xlabel('Sport') plt.ylabel('Number of Votes') plt.show()
Python — Horizontal Bar Chart
import matplotlib.pyplot as plt sports = ['Soccer', 'Basketball', 'Baseball', 'Tennis', 'Swimming'] votes = [12, 9, 6, 4, 7] plt.barh(sports, votes, color='darkorange') plt.title('Favorite Sports in Our Class') plt.xlabel('Number of Votes') plt.show()

🔑 Key Parameters

colorFill color of bars — single color or a list of colors for each bar
edgecolorBorder color around each bar
widthWidth of each bar (default 0.8). Use height for barh()
💡 Tip: Use plt.barh() (horizontal) when category names are long — they are easier to read on the y-axis.

🥧 Pie Chart

A pie chart shows each category as a slice of a circle. The size of each slice represents the proportion (percentage) of the whole. The full circle always equals 100%.

When to use: You want to show how a total is divided among categories — "What fraction of the whole does each category represent?" Works best with 2–5 categories.
Python
import matplotlib.pyplot as plt # Favorite music genres genres = ['Pop', 'Hip Hop', 'Rock', 'Country', 'K-Pop'] counts = [15, 10, 5, 3, 7] plt.pie(counts, labels=genres, autopct='%1.1f%%', colors=['#ff9999', '#66b3ff', '#99ff99', '#ffcc99', '#c2c2f0']) plt.title('Favorite Music Genres') plt.show()

🔑 Key Parameters

labelsList of category names shown next to each slice
autopctFormat string for showing percentages on each slice (e.g., '%1.1f%%')
colorsList of colors, one per slice
explodeTuple to "pull out" a slice, e.g., (0.1, 0, 0, 0, 0) pulls the first slice
⚠️ Caution: Pie charts become hard to read with more than 5–6 categories. When in doubt, use a bar chart instead!

Quick Reference

ChartFunctionBest For
Vertical Barplt.bar()Comparing counts across categories
Horizontal Barplt.barh()Same as above, better for long labels
Pie Chartplt.pie()Part-of-a-whole, proportion of total
