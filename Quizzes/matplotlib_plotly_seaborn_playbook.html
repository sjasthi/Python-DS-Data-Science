Matplotlib, Seaborn & Plotly

An interactive overview & comparison of Python's top visualization libraries

Review

Matplotlib — The Foundation

Matplotlib is Python's original plotting library, created in 2003. It gives you complete control over every element of your chart. Think of it as the "building blocks" that other libraries are built on.

Key Things to Remember

  • Static plots — creates images (PNG, PDF, SVG), not interactive web pages
  • Fine-grained control — you decide every color, label, tick mark, and line width
  • The foundation — Seaborn is built directly on top of Matplotlib
  • Great for reports — publication-quality figures for papers and homework

New

Seaborn — Beautiful Stats Made Easy

Seaborn is built on top of Matplotlib and was created by Michael Waskom in 2012. It makes statistical plots look beautiful with very little code. It works hand-in-hand with pandas DataFrames.

Why Seaborn?

  • Gorgeous defaults — charts look polished without extra styling work
  • Statistical superpowers — built-in box plots, violin plots, pair plots, heatmaps
  • DataFrame-friendly — just pass column names, no extracting needed
  • Built on Matplotlib — you can still use plt.title(), plt.xlabel(), etc.

Key Relationship

🔑 Important: Seaborn is NOT a replacement for Matplotlib — it is a layer on top of it. Every Seaborn chart is a Matplotlib figure underneath. You still call plt.show() to display it!

Seaborn Scatter Plot

Python — Seaborn📓 Run in Colab
import seaborn as sns
import matplotlib.pyplot as plt

# Seaborn comes with built-in datasets!
tips = sns.load_dataset('tips')

sns.scatterplot(data=tips, x='total_bill', y='tip', hue='day')
plt.title('Tips by Total Bill')
plt.show()

Seaborn Box Plot

Python — Seaborn📓 Run in Colab
import seaborn as sns
import matplotlib.pyplot as plt

tips = sns.load_dataset('tips')

sns.boxplot(data=tips, x='day', y='total_bill', palette='Set2')
plt.title('Total Bill by Day of Week')
plt.show()

Seaborn Heatmap (Correlation Matrix)

Python — Seaborn📓 Run in Colab
import seaborn as sns
import matplotlib.pyplot as plt

tips = sns.load_dataset('tips')

sns.heatmap(tips.corr(numeric_only=True), annot=True, cmap='coolwarm')
plt.title('Correlation Heatmap')
plt.show()

Seaborn Pair Plot — The One-Liner Wonder

Python — Seaborn📓 Run in Colab
import seaborn as sns

iris = sns.load_dataset('iris')

# One line → a grid of ALL pairwise relationships!
sns.pairplot(iris, hue='species')
plt.show()

⬆️ This one line creates a full grid of scatter plots and histograms for every pair of columns. That would take dozens of lines in Matplotlib!

New

Plotly — Interactive & Web-Ready

Plotly is a modern graphing library released in 2012. Unlike Matplotlib and Seaborn, it creates interactive charts using JavaScript. You can hover over data points, zoom in, pan around, and export charts as HTML files.

Why Plotly?

  • Interactive by default — hover tooltips, zoom, pan, and click right out of the box
  • Web-ready — charts are rendered in the browser using JavaScript (D3.js & Plotly.js)
  • Great 3D support — rotate and explore 3D scatter plots and surfaces
  • Plotly Express — a simple, high-level API (similar to Seaborn's ease-of-use)
  • Shareable — save charts as standalone HTML files anyone can open

Key Difference from Matplotlib & Seaborn

🔑 Important: Plotly is a completely separate library — it does NOT use Matplotlib under the hood. It uses its own JavaScript-based rendering engine. That's what makes it interactive!

Plotly Express Scatter Plot

Python — Plotly Express📓 Run in Colab
import plotly.express as px

df = px.data.iris()

fig = px.scatter(df, x='sepal_width', y='sepal_length',
                 color='species',
                 title='Iris: Sepal Width vs Length')
fig.show()

⬆️ Hover over any point to see its exact values. Try zooming with your scroll wheel!

Plotly Express Bar Chart

Python — Plotly Express📓 Run in Colab
import plotly.express as px

df = px.data.tips()

fig = px.bar(df, x='day', y='total_bill', color='sex',
             barmode='group',
             title='Total Bill by Day and Gender')
fig.show()

Plotly Express Line Chart

Python — Plotly Express📓 Run in Colab
import plotly.express as px

df = px.data.gapminder().query("country == 'United States'")

fig = px.line(df, x='year', y='gdpPercap',
              title='US GDP Per Capita Over Time')
fig.show()

Plotly 3D Scatter Plot

Python — Plotly Express📓 Run in Colab
import plotly.express as px

df = px.data.iris()

fig = px.scatter_3d(df, x='sepal_length', y='sepal_width',
                    z='petal_length', color='species',
                    title='3D Iris Dataset')
fig.show()

⬆️ In Colab, you can click and drag to rotate this 3D plot in any direction!

Quick Decision Guide

I want to…Use…
Create a static figure for a reportMatplotlib
Quickly explore data distributionsSeaborn
Make a chart users can hover & zoomPlotly
Build a correlation heatmapSeaborn
Create a 3D plot I can rotatePlotly
See ALL pairwise relationships at onceSeaborn (pair plot)
Share an interactive chart as a filePlotly (HTML)
Have pixel-level control over a plotMatplotlib

plt.scatter(df['x'], df['y'], c='blue')

sns.scatterplot(data=df, x='x', y='y')

fig = px.scatter(df, x='x', y='y', title='Scatter Plot')
