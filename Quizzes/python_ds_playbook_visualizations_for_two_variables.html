Why Two Variables?

Moving beyond single-variable charts

Last week you learned how to visualize one variable at a time — histograms for quantitative data, bar charts for qualitative data. This week we ask a bigger question: how are two variables related to each other?

The right chart depends on what types of variables you are comparing:

Quant vs QuantTwo numerical variables — e.g., height vs weight, hours studied vs test score
Quant vs QualA numerical variable split by a category — e.g., test scores by grade level
Qual vs QualTwo categorical variables — e.g., favorite sport vs gender

⬡ Scatter Plot

A scatter plot places one variable on the x-axis and another on the y-axis. Each data point becomes a dot. The pattern of dots reveals whether the variables are positively correlated, negatively correlated, or have no clear relationship.

When to use: Both variables are quantitative and you want to see if a relationship (correlation) exists — e.g., "Do taller students weigh more?"
Python — Scatter Plot

🔑 Key Parameters

sSize of the dots (default ~36). Larger values make dots bigger.
colorColor of the dots (e.g., 'dodgerblue', '#e74c3c')
alphaTransparency from 0 (invisible) to 1 (solid). Useful when dots overlap.
markerDot shape — 'o' circle, 's' square, '^' triangle, 'x' cross

📈 Line Chart (Quant vs Quant over time)

When the x-axis variable is time (or any ordered sequence), a line chart connects the dots to highlight trends. You can overlay multiple lines to compare two quantitative series.

When to use: One quantitative variable changes over time and you want to show the trend — e.g., "How did temperature and rainfall change each month?"

📊 Grouped Bar Chart

A grouped bar chart places bars for different categories side by side, making it easy to compare a quantitative measure across groups. You control bar positions using numerical x-values and offsets.

When to use: You want to compare average values (or totals) of a quantitative variable across different categories — e.g., "Average test score by grade level."
Python — Grouped Bar Chart

🔑 Key Technique

Use np.arange() to create numerical x-positions, then offset each group by ±width/2. Call plt.xticks() to replace the numbers with category labels.

📦 Side-by-Side Box Plot

Placing multiple box plots next to each other lets you compare the distribution of a quantitative variable across different categories — including median, spread, and outliers for each group.

When to use: You want to compare the full distribution (not just averages) of a quantitative variable across categories — e.g., "How do test score distributions differ by class period?"
Python — Side-by-Side Box Plot

🔑 Key Technique

Pass a list of lists to plt.boxplot() — each inner list becomes its own box. Use the labels parameter to name the categories.

📊 Grouped Bar Chart (two categorical variables)

When both variables are categorical, a grouped bar chart shows counts for each combination of categories side by side. One category goes on the x-axis; the other is represented by different colored bars.

When to use: You want to compare counts across every combination of two categorical variables — e.g., "How many boys vs girls chose each favorite sport?"
Python — Grouped Bar (Qual vs Qual)

📊 Stacked Bar Chart

A stacked bar chart places one group's bars on top of the other, so each bar shows the total and you can see each group's contribution. Use the bottom parameter to stack.

When to use: You want to show totals AND the breakdown within each category — e.g., "Total votes per sport, split by gender."
Python — Stacked Bar Chart

🔑 Key Parameter

bottomA list of values that defines where each bar starts. Set it to the first group's data to stack the second group on top.
